ทำความรู้จัก Flask

Flask คือ micro web framework ที่เขียนด้วย python. Flask ถูกจัดให้อยู่ใน microframework เพราะว่าไม่ต้องการใช้เครื่องมือหรือไลบารี่เฉพาะ ดังนั้นจึงมีแค่ที่จำเป็นเลยเล็ก แต่เขียนไม่กี่บรรทัดก็รันได้เลย Flask ไม่มีทั้ง database หรือการตรวจการตรวจแบบฟอร์ม (form validation) แต่ยังรองรับ extensions ที่ช่วยเพิ่มความสามารถอื่นได้ ถ้ารองรับการใช้กับ Flask ตัวอย่างเว็บ ที่นำ Flask ไปใช้ คือ Pinterest LinkedIn เป็นต้น

Installation for start Flask project

  • ติดตั้ง Python version 3.6 or Newer
  • Create Virtual environments 
    • $ mkdir flaskproject
$ cd flaskproject
$ python -m venv fw

    fw คือชื่อของ Virtual Environment ของโปรเจคท์นี้

    note: สำหรับ macOS/ linux ให้ใช้คำสั่ง python3 -m venv env

  • Activate Virtual environments
    • $ fw\scripts\activate

    note2: สำหรับ macOS/ linux ให้ใช้คำสั่ง . env/bin/activate

  • ติดตั้ง Flask
    • $ pip install flask

เป็นอันเสร็จสิ้นขั้นตอนนี้

สร้างไฟล์ app.py ทำการเขียนโค้ดดังนี้ 


from flask import Flask
                                    
app = Flask(__name__)
                                                    
@app.route("/")
def hello_world():
    return "Hello My First Flask Project"

    app.run(debug=True)

อธิบายโค้ด

from flask import Flask

บรรทัดแรกจะเป็นการอิมพอร์ต Flask จากโมดูล flask เพื่อสร้าง Flask เว็บเซิร์ฟเวอร์ของ Flask App จากโมดูล Flask

app = Flask(__name__)

คำสั่งนี้จะใช้บ่งบอกว่า ไฟล์นี้คือไฟล์ Main ที่จะเอาไว้รัน Flask App ของเรานั่นเอง 

@app.route("/")
def hello_world():
    return "
Hello, World!
"

ส่วนต่อมาจะเป็นการสร้าง URL เพื่อทำการชี้หรือ Map ไปที่ URLs ที่ต้องการ โดยในที่นี้ จะเป็นหน้า URL เริ่มต้น ซึ่งส่วนใหญ่หรือแทบจะทุกเว็บจะใช้ URL นี้เป็นหน้าโฮมเพจหรือที่อยู่ของเว็บไซต์

if __name__ == '__main__':
    app.run(debug=True)

ส่วนสุดท้ายจะเป็นการรัน Flask และทำการเปิด Debug Mode เพื่อใช้สำหรับติดตาม Error ต่าง ๆ ได้

ปล. เวลา Deploy ให้ปิด Debug Mode โดยใช้ debug=False

ทดลองรันโค้ดข้างต้น

$ flask run

โดยตัว Default ที่ผมกำลังรันอยู่ในตอนนี้คือ localhost หรือไอพีแอดเดรส 127.0.0.1 พอร์ต Default ของ Flask คือ 5000 สุดท้ายเมื่อรันแล้วทำการเปิดหน้าเว็บบราวเซอร์จะพบ 127.0.0.1:5000

เมื่อเราเข้าไปที่ http://127.0.0.1:5000 หรือ http://localhost:5000/ ก็ได้หน้าเว็บมาแล้วเรียบร้อย และนี่ก็คือหน้าเว็บจริง ๆ ครับ โดยในหน้าเว็บนี้แสดงผลคำว่า “Hello My First Flask Project” ที่ผมได้เขียนฟังก์ชันจากนั้นทำการรีเทิร์น ข้อความนี้ออกมาที่ URL ที่เป็นหน้า Default คือหน้าเริ่มต้นของเว็บ ซึ่งปกติเราจะใช้หน้านี้เป็นหน้าโฮมเพจ

http://127.0.0.1:5000

การเพิ่ม route และใส่ค่าลงไปใน url ได้เพื่อนำค่านั้นไปใช้

เพิ่ม code ส่วนนี้เข้าไป ใน app.py

@app.route('/hello/')
def hello_name(name):
    return "hello {}. ".format(name)

จะได้


from flask import Flask
                                                            
app = Flask(__name__)
                                                                            
@app.route("/")
def hello_world():
    return "Hello My First Flask Project"

@app.route('/hello/< name>')
def hello_name(name):
    return "hello {}. ".format(name)
                        
if __name__ == '__main__':
    app.run(debug=True)

แล้ว ลอง http://localhost:5000/hello/ชื่อ หรือ ข้อความที่่ต้องการให้แสดง ก็จะได้

http://127.0.0.1:5000/hello/Film

การใช้งาน render_template เพื่อใช้ html

สร้าง folder ที่ชื่อว่า templates ใช้เก็บไฟล์ที่ใช้ในการแสดงผลเป็นหน้าเว็บ (html )

create templates

เข้าไปใน templates

สร้างไฟล์ home.html ซึ่งจะเป็นหน้าเว็บของเรา


<html>
    <head>
        <title> {{ title }}</title>
    </head>
    <body>                  
        <h1> you may already know this is home page </h1>               
    </body>
</html>

และ hello.html หน้าสวัสดีชื่อของเรา


<html>
    <head>
        <title> {{ title }}</title>
    </head>
    <body>                  
        <h1> hello I know you! you are {{name}} </h1>               
    </body>
</html>

และแก้ใน app.py เพิ่ม render_template และใช้มัน render หน้าเว็บขึ้นมา และส่งตัวแปรไปด้วย


from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('home.html',title='home')

@app.route('/hello/')
def hello_name(name):
    return render_template('hello.html',title='hello_name',name=name)

                                                
if __name__ == '__main__':
    app.run(debug=True)

ซึ่งเป็นค่าที่ส่งมาจากแอปหลักในการ render_template นั้นเอง

หลังจากนี้เราก็สามารถปรับแต่งหน้าเว็บเราในไฟล์ template ได้เลยแล้วให้โปรแกรม render template ออกมา

มาเริ่มลุย CSS กันเลยครับ

สร้าง folder ที่ชื่อว่า static/css ทำการสร้างไฟล์ CSS ขึ้นมา 1 ไฟล์ ในโฟลเดอร์ static/css โดยตั้งชื่อว่า style.css

create style.css

ในไฟล์ style.css ใส่ style ที่เราต้องการ


body{
    background-color: darksalmon;
}
h1{
    color: white;
}

และ แก้ไขไฟล์ home.html เพื่อทำการ import ไฟล์ css ของเรามาใช้

<link rel="stylesheet" href="{{ url_for ('static', filename='css/style.css')}}">

<html>
    <head>
        <link rel="stylesheet" href="{{ url_for ('static', filename='css/style.css')}}">
        <title> {{ title }}</title>
    </head>
    <body>                  
        <h1> you may already know this is home page </h1>               
    </body>
</html>
